# Инструкция по загрузке данных

## Датасет: Smart Meters in London

### Шаги для загрузки:

1. Зарегистрируйтесь на [Kaggle](https://www.kaggle.com/)
2. Перейдите на страницу датасета: [Smart meters in London](https://www.kaggle.com/datasets/jeanmidev/smart-meters-in-london)
3. Нажмите кнопку "Download" для скачивания датасета
4. Распакуйте архив в эту директорию (`data/raw/`)

### Структура данных:

Датасет содержит следующие файлы:
- `block_*.csv` - файлы с почасовыми данными потребления для каждого блока домохозяйств
- `informations_households.csv` - информация о домохозяйствах
- `informations_weather.csv` - данные о погоде

### Альтернативный вариант (для тестирования):

Если датасет слишком большой, можно использовать упрощенный вариант:
- Создать тестовый CSV файл с колонками: `LCLid`, `day`, `energy_kwh`, `timestamp`

Пример структуры CSV:
```csv
LCLid,day,energy_kwh,timestamp
MAC000002,2011-11-01,0.123,2011-11-01 00:00:00
MAC000002,2011-11-01,0.456,2011-11-01 01:00:00
```
